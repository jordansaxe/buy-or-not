# Buy-or-Not Calculator

A tiny tool to decide smarter: resale offsets, wait-for-sale sim, per-use math, minimalism nudge, and saved history (localStorage).

## Tech

- React + Vite
- Tailwind CSS
- TypeScript-friendly
- Deployed on Vercel

## Commands (local dev)

```bash
npm install
npm run dev
npm run build
npm run preview
```
